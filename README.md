Blood Sample Management System

This is a backend project developed using Java and Springboot for managing home blood sample collection and appointment booking. 

Features:
- User Registration and Login
- Jwt based user authentication with User and Admin role access
- Book multiple lab tests in one appointment with auto total price calculation
- Smart Technician auto assignment based on city and daily visit limit (max 6 per day)
- Duplicate booking preventation
- Admin manages test packages, technicians, sample collection, payment status and report link upload
  
Technologies used: Java, Springboot, Spring Security, Spring Data JPA, Mysql

Tools used: IntelliJ, Postman, MySQL WorkBench, Git
